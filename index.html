Aide

Cette page est conçue pour être utilisée avec VoiceOver sur iPhone. Balaye l’écran pour entendre les sections : date, prochaine prière, liste des prières, actions vocales, paramètres.

Le bouton « Annoncer la prochaine prière » lit uniquement la prochaine prière. Le bouton « Lire tous les horaires » lit toutes les prières du jour.

Le message d’amour peut être activé ou désactivé dans les paramètres ci-dessus.
